# Pok-Deng Game CLI

## Installation

1.  Clone the repository.

    ```bash
    git clone git@github.com:danyouknowme/pok-deng-game-cli.git && cd pok-deng-game-cli
    ```

2.  Install the dependencies.

    ```bash
    npm install
    ```

3.  Run unit tests.

    ```bash
    npm run test
    ```

4.  Start game in development.

    ```bash
    npm run dev
    ```

5. Build.

    ```bash
    npm run build
    ```

6. Start game.
     ```bash
     npm run start
     ```
